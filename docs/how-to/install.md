# Install the SDK

## As a standalone package

```bash
pip install owi-metadatabase-shm
```

Using `uv`:

```bash
uv pip install owi-metadatabase-shm
```

## As a core-package extra

```bash
pip install "owi-metadatabase[shm]"
```

Using `uv`:

```bash
uv pip install "owi-metadatabase[shm]"
```

## Development setup

```bash
git clone https://github.com/OWI-Lab/owi-metadatabase-shm-sdk.git
cd owi-metadatabase-shm-sdk
uv sync --dev
```
