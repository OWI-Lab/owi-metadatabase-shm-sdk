# How-to Guides

How-to guides are **problem-oriented** — each one solves a specific task
with a minimum of background explanation.

| Guide | What it covers |
|-------|----------------|
| [Install the SDK](install.md) | Install standalone or as a core-package extra. |
| [Authenticate](authenticate.md) | Obtain and configure your API token. |
| [Get Sensor Data](get-sensor-data.md) | Read sensor types, sensors, and calibrations. |
| [Get Signal Data](get-signal-data.md) | Read signals, histories, calibrations, and derived signals. |
| [Upload Sensor Types](upload-sensor-types.md) | Upload sensor type records with optional images. |
| [Upload Sensors](upload-sensors.md) | Upload per-turbine sensor records by category. |
| [Upload Sensor Calibrations](upload-sensor-calibrations.md) | Upload calibrations with optional PDF datasheets. |
| [Process Signal Configs](process-signal-configs.md) | Parse JSON config events into typed signal records. |
